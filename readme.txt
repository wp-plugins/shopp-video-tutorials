== Description ==

This plugin provides video tutorials and online lessons for the [Shopp e-commerce plugin](http://shopplugin.net).

== Installation ==

1. Ensure that your [Shopp](http://shopplugin.net) is activated and up-to-date.
2. Download Shopp + Video Tutorials. Unzip the `shopp-video-tutorials` folder.
3. Upload the `shopp-video-tutorials` folder to the `wp-content/plugins/` directory.
4. Activate Shopp + Video Tutorials through the Plugins menu in the WordPress admin.
5. Go to Shopp 101 &rarr; + Video Tutorials.
